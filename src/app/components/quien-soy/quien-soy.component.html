Explicación del juego propio

Nombre del juego: Cálculo veloz

El juego creará y mostrará al usuario operaciones matemáticas de manera aleatoria. Se debe resolver la operación y escribir el resultado en un input, antes de que se acabe el tiempo. Cada vez que el usuario responda correctamente, se le sumara puntos. Si el usuario responde incorrectamente, se le restará una vida. Cada usuario tiene 3 vidas. Si las vidas se acaban, el juego termina y se mostrará la cantidad de puntos obtenidos.
